# Arduino

Arduino LCD Oled + I2C on 2 devices ready to use code.

### How to

This project(s) uses PlatformIO. http://platformio.org/
Choose board 

[env:nanoatmega328]
platform = atmelavr
board = nanoatmega328
framework = arduino

Check if the libraries are ok
Hit RUN
